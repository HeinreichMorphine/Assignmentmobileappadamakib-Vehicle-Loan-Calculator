# Vehicle Loan Calculator

This is a simple Android application that helps users calculate their vehicle loan payments.

## Features

*   **Loan Calculation:** Calculates the monthly loan payment, total payment, and total interest based on the vehicle cost, down payment, interest rate, and loan term.
*   **About Page:** Provides information about the application.

## How to Use

1.  Enter the vehicle cost, down payment amount, annual interest rate, and loan term in years.
2.  The application will automatically calculate and display your estimated monthly payment, total payment, and total interest paid.
3.  Navigate to the "About" screen to learn more about the app.
